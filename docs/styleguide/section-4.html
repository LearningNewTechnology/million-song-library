4 #permalink Helpers

4.1 #permalink Clear

Simple class for setting clear: both on an element.

Note: this is different from .clearfix, which is meant to 'self-clear' an element's floated children.

Example

Lorem ipsum

<p class="clear">Lorem ipsum</p>
