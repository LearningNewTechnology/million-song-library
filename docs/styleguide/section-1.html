1 #permalink Variables

1.1 #permalink Colors - Raw

These are the lowest level color variables.

  • $orange-1 #FF9800
  • $orange-2: #E65100
  • $blue-1: #2196F3
  • $blue-2: #0D47A1
  • $blue-3: #263238
  • $black: #000000
  • $gray-1: #777777
  • $gray-2: #CCCCCC
  • $gray-3: #E7E7E7
  • $gray-4: #F8F8F8
  • $white: #FFFFFF

1.2 #permalink Colors - Semantic

Semantic color variables map to both generic and raw color variables, and allow us to name values based on their intended application.

  • $color-text: $black
  • $color-text-grey: $gray-1

1.3 #permalink Typography
